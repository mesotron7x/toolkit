install-windows-devbox.ps1

Windows

Bootstraps a Windows development host for SSH access and basic CLI tooling. It installs OpenSSH Server, Google Chrome, Scoop, Git, and Vim, sets the OpenSSH default shell to the built-in Windows PowerShell 5, and prepares the administrators authorized keys file. If launched from PowerShell 7 (pwsh), it relaunches itself in Windows PowerShell 5.1 before installing Windows optional features such as OpenSSH Server.

irm https://raw.githubusercontent.com/mesotron7x/toolkit/main/install-windows-devbox.ps1 | iex

What it does

  • Starts installing Windows OpenSSH Server in the background while Chrome and Scoop tools install.
  • Relaunches itself in Windows PowerShell 5.1 when needed so OpenSSH optional-feature commands run through the Windows PowerShell DISM cmdlets.
  • Starts and enables the sshd service.
  • Allows TCP/22 through Windows Defender Firewall.
  • Installs Google Chrome through winget.
  • Installs Scoop, Git, and Vim.
  • Sets OpenSSH's default shell to Windows PowerShell 5.
  • Creates C:\ProgramData\ssh\administrators_authorized_keys with strict permissions.

Safety note

The irm ... | iex pattern is convenient, but it executes remote code immediately. For a more auditable flow, open the installer source first, review it, and then run it.
